Password Cube - Privacy Policy

Last updated: April 2025

1. Introduction

Password Cube is a Chrome extension designed to generate and manage secure passwords. Your privacy is important to us, and this document outlines how Password Cube handles data.

2. Data Collection & Storage

Root Domain: Password Cube stores the root domain of websites for which passwords are generated and managed.

Expiration Date: The expiration date for the generated passwords is stored locally to manage when passwords should be renewed or updated.

Local Storage: This data (root domain and expiration date) is securely stored in Chrome's local storage and is never shared or transmitted to external servers.

3. Third-Party Services

Password Cube does not interact with third-party services for data collection. The extension operates entirely offline, with no reliance on external APIs.

4. Security Measures

  • All data (root domain and expiration date) is stored securely in Chrome's local storage.
  • Password Cube does not transmit any data to external servers, ensuring your data stays private.

5. No Data Sharing or Advertising

Password Cube does not collect, share, or sell user data. There are no advertisements, trackers, or analytics within the extension. All data remains securely stored on your device.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page.
